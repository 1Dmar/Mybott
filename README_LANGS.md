# 🌍 نظام اللغات المتعددة (Multi-Language System)

لقد قمت بتفعيل وتطوير نظام اللغات في البوت ليعمل بشكل تلقائي لكل سيرفر.

## 🚀 كيف يعمل النظام؟

1.  **دالة الترجمة المركزية**: تم إضافة دالة `client.t(guildId, key)` في ملف `bot/index.js`.
2.  **التخزين المؤقت (Caching)**: يتم تحميل لغات جميع السيرفرات عند تشغيل البوت وتخزينها في الذاكرة لضمان سرعة الاستجابة دون الضغط على قاعدة البيانات.
3.  **التحديث الفوري**: عند تغيير اللغة باستخدام أمر `/setlanguage` يتم تحديث الذاكرة فوراً وتتغير لغة البوت في السيرفر مباشرة.

## 🛠️ كيف تستخدمه في الأوامر الجديدة؟

بدلاً من كتابة النصوص مباشرة، استخدم الدالة كالتالي:

```javascript
// مثال في أي أمر
run: async (client, interaction) => {
    const guildId = interaction.guild.id;
    const message = client.t(guildId, "WELCOME_MESSAGE");
    
    await interaction.reply({ content: message });
}
```

## 📝 كيف تضيف ترجمات جديدة؟

1.  افتح ملف `bot/public/json/translations.json`.
2.  أضف المفتاح الجديد (Key) تحت كل لغة (`en`, `ar`, `zh`).
3.  تأكد من استخدام نفس المفتاح في جميع اللغات.

## ⚙️ أمر تغيير اللغة
الأمر هو `/setlanguage` (موجود في `bot/Commands/Slash/Server/language`).
لقد قمت بتحديثه ليدعم التحديث الفوري للكاش.

---
**ملاحظة**: قمت بتطبيق هذا النظام كمثال في أمر `/help` لترى كيف يعمل. يمكنك الآن تطبيقه على بقية الأوامر بنفس الطريقة.
