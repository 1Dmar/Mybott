# 📁 نظام الرموز التعبيرية المركزي (Emoji System)

لقد قمت بتنظيم جميع الرموز التعبيرية المستخدمة في البوت داخل ملف واحد لتسهيل عملية التعديل عليها بضغطة زر واحدة.

## 🚀 كيف تغير الرموز التعبيرية؟

1. افتح الملف الموجود في المسار التالي:
   `bot/settings/emojis.js`

2. ستجد قائمة بجميع الرموز التعبيرية مصنفة حسب الاستخدام، مثال:
   ```javascript
   SUCCESS: "✅",
   ERROR: "❌",
   WARNING: "⚠️",
   ```

3. قم بتغيير الرمز الموجود بين العلامتين `" "` إلى الرمز الذي تريده.

4. احفظ الملف وأعد تشغيل البوت، وسيتم تحديث الرمز في جميع الرسائل والأوامر تلقائياً!

## 🛠️ كيف تم الربط؟

- تم إضافة الرموز إلى كائن العميل `client.emojis` في ملف `bot/index.js`.
- تم تحديث أغلب ملفات الأوامر والأحداث لتستخدم `${client.emojis.KEY}` بدلاً من الرموز المباشرة.

---
**ملاحظة:** إذا كنت تستخدم الرموز في ملفات جديدة، تأكد من استخدام `client.emojis.اسم_الرمز` للوصول إليها.
